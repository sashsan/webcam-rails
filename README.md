# README

Environment:

* `Ruby 2.6.3`
* `Rails 6.0.2.1`
* `Node JS 13.9.0`
* `Yarn 1.21.1`
* `PostgreSQL 11`

Setup project

* `git clone https://github.com/sashsan/webcam-rails.git`
* `cd webcam-rails`
* `bundle exec rake db:create`
* `bundle exec rake db:migrate`
* `bundle exec rake db:seed`
* `http://localhost:3000/`

Tests Rspec

* `bundle exec rspec spec`
